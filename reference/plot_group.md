# Map a plot-type key to its aspect-ratio group

Map a plot-type key to its aspect-ratio group

## Usage

``` r
plot_group(plot_type)
```

## Arguments

- plot_type:

  Character; plot-type key (e.g. `"period"`, `"phase_distribution"`,
  `"spatial_clusters"`).

## Value

Character; one of `"square"`, `"tall"`, `"spatial"`, or
`"network_spatial"`.
