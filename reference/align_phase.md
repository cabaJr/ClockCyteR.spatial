# Align a vector of phases to a target Circadian Time

Align a vector of phases to a target Circadian Time

## Usage

``` r
align_phase(data, CT = 6, ...)
```

## Arguments

- data:

  Numeric vector of phases in radians.

- CT:

  Numeric; target Circadian Time in hours. Defaults to `6`.

- ...:

  Currently unused.

## Value

Numeric vector of aligned phases in radians.
