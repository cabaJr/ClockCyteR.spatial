# Create a subdirectory if it does not already exist

Create a subdirectory if it does not already exist

## Usage

``` r
create_folder(base_dir, folder_name)
```

## Arguments

- base_dir:

  Character; parent directory path.

- folder_name:

  Character; name of the subdirectory to create.

## Value

Character; full path to the (now existing) subdirectory.
