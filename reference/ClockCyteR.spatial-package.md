# ClockCyteR.spatial: ClockCyteR.spatial implements spatial analysis of circadian data

The ClockCyteR.spatial package enables spatiotemporal analysis of
timeseries data extracted from timelapse images across multiple
channels. It performs multivariate analysis of circadian parameters
starting from period, phase, and amplitude from defined ROIs and can
calculate network features (optimised for Suprachiasmatic Nucleus
organotypic slices).
