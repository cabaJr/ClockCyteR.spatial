# Create an empty list to accumulate FFT analysis results

Create an empty list to accumulate FFT analysis results

## Usage

``` r
createFFTtbl()
```

## Value

A list with empty vectors for each result field: `variance_vec`,
`analysed_names_vec`, `reporter_vec`, `traces_no_vec`, `tot_traces`,
`vector_len`, `period_var`, `period_mean`, `amplitude_mean`,
`error_mean`.
